# Тест аккаунта: вход и отправка сообщения

## 1. Запуск проекта

**Важно:** закройте все старые окна, где уже был запущен FireNotify (чёрные окна с логами). Иначе порт 3001 будет занят и backend не запустится.

Затем дважды нажмите **ZAPUSTIT_VSE.bat** в корне папки POzharApp.

В консоли должны появиться:
- `FireNotify API running at http://localhost:3001`
- `Local: http://localhost:5173` (или 5174, 5175… — **этот адрес откройте в браузере**)

---

## 2. Открыть приложение и войти

1. Откройте в браузере: **http://localhost:5173** (или тот порт, что показан в консоли).
2. На странице входа введите:
   - **Логин и пароль:** заданы в `.env` (ADMIN_LOGIN, ADMIN_PASSWORD) или выведены в консоль при первом запуске `npx prisma db seed`.
3. Нажмите **«Войти»**.

Должна открыться главная панель (Dashboard).

---

## 3. Отправить сообщение

1. В меню слева нажмите **«Отправить сообщение»**.
2. **Получатель:** выберите администратора (если один — будет только admin).
3. **Тема:** например, «Тестовое сообщение».
4. **Текст сообщения:** например, «Проверка отправки».
5. Нажмите **«Отправить»**.

Должно появиться сообщение об успешной отправке.

---

## 4. Проверить (опционально)

- **Уведомления** — список пожарных уведомлений (можно создать новое для проверки).
- **Настройки** — привязка Telegram (запросить код, отправить боту `/start КОД`).

Логин и пароль — из `.env` (ADMIN_LOGIN, ADMIN_PASSWORD) или сгенерированы при seed (см. вывод консоли).
