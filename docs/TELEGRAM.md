# Интеграция Telegram

В системе уведомления доставляются **только через Telegram** (Push, SMS, Email удалены).

## Токен уже в .env — что делать дальше

1. **Установить зависимости:** в папке `backend` выполнить `npm install`.
2. **Перезапустить backend** (из корня: `npm run dev`). В логах должно быть: `[Telegram] Bot started: @ваш_бот`.
3. Открыть приложение (http://localhost:5173), войти (логин и пароль из .env или выведенные при seed).
4. **Настройки** → включить «Включить уведомления в Telegram» → **Запросить код привязки**.
5. В Telegram открыть бота и отправить: **`/start ВАШ_КОД`** (код из шага 4).
6. Бот ответит «Аккаунт привязан...». В приложении нажать **Сохранить настройки**.

Подробная инструкция: **TELEGRAM_ЧТО_ДЕЛАТЬ.md** в корне проекта.

---

## Что сделано

1. **Привязка аккаунта** — пользователь запрашивает код в настройках, отправляет боту команду `/start КОД`, аккаунт привязывается к Telegram.
2. **Уведомления** — при создании нового пожарного уведомления все администраторы с включённым Telegram получают сообщение в бот.
3. **Отвязка** — в настройках кнопка «Отвязать Telegram».

## Настройка бота

1. Откройте в Telegram [@BotFather](https://t.me/BotFather).
2. Отправьте `/newbot`, укажите имя и username бота.
3. Скопируйте выданный **токен** (вид: `123456789:ABCdefGHI...`).
4. В папке `backend` откройте файл `.env` (или создайте из `env.example.txt`).
5. Добавьте строку:
   ```
   TELEGRAM_BOT_TOKEN=ваш_токен_сюда
   ```
6. Перезапустите backend (`npm run dev` в корне или в `backend`).

После перезапуска в логах должно появиться: `[Telegram] Bot started: @ваш_бот`.

## Как пользоваться

1. Войдите в систему (логин и пароль администратора из .env или seed).
2. Откройте **Настройки**.
3. Включите «Уведомления через Telegram».
4. Нажмите **Запросить код привязки**.
5. В интерфейсе появится код и ссылка на бота. Откройте бота в Telegram и отправьте: `/start ВАШ_КОД`.
6. Бот ответит «Аккаунт привязан...».
7. Сохраните настройки. Готово — при новых пожарных уведомлениях вы получите сообщение в Telegram.

## Файлы в проекте

- `backend/src/telegram/bot.ts` — инициализация бота, команда `/start`, отправка сообщений.
- `backend/src/lib/telegramNotify.ts` — рассылка уведомлений админам при новом сигнале.
- `backend/src/routes/settings.ts` — запрос кода привязки, отвязка.
- `backend/src/routes/notifications.ts` — при создании уведомления вызывается `notifyTelegramNewNotification`.

Если `TELEGRAM_BOT_TOKEN` не задан, бот не запускается, привязка и уведомления в Telegram недоступны (остальное работает).
