# Создание заявок через e-mail (соответствие ТД)

Тендерная документация предусматривает: **возможность создания заявок через e-mail**, SMTP/IMAP интеграцию, автоматическое преобразование входящего письма в заявку.

## Реализация

В системе FireNotify реализован приём заявок по e-mail через **интеграцию с IMAP**:

- Подключение к почтовому ящику по протоколу IMAP.
- Периодический опрос ящика (скрипт или cron).
- Каждое новое (непрочитанное) письмо преобразуется в заявку (уведомление):
  - **Источник:** «Email»
  - **Адрес/тема:** тема письма
  - **Описание:** отправитель и текст письма

## Настройка

В файле `backend/.env` задаются переменные:

```env
IMAP_HOST=imap.example.com
IMAP_PORT=993
IMAP_SECURE=true
IMAP_USER=inbox@example.com
IMAP_PASSWORD=пароль_или_пароль_приложения
IMAP_MAILBOX=INBOX
```

## Запуск

- **Вручную (из папки backend):**  
  `npm run email:ingest`

- **По расписанию (cron на сервере), например каждые 5 минут:**  
  `*/5 * * * * cd /opt/firenotify/backend && npm run email:ingest`

Если переменные IMAP не заданы, скрипт завершается без ошибки (интеграция отключена).
