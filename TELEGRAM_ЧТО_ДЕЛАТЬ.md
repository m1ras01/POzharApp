# Токен бота в .env добавлен — что делать дальше

## Шаг 1. Установить telegraf (если ещё не ставили)

В терминале:

```bash
cd c:\Users\Алекс\Desktop\POzharApp\backend
npm install
```

(Пакет `telegraf` уже есть в `package.json`, он установится вместе с остальными.)

---

## Шаг 2. Перезапустить backend

Остановите текущий запуск (Ctrl+C в терминале, где идёт `npm run dev`), затем снова:

```bash
cd c:\Users\Алекс\Desktop\POzharApp
npm run dev
```

В логах должно появиться что-то вроде: **`[Telegram] Bot started: @ваш_бот`**  
Если этой строки нет — проверьте, что в `backend/.env` строка `TELEGRAM_BOT_TOKEN=...` без пробелов и кавычек вокруг токена.

---

## Шаг 3. Открыть приложение и войти

В браузере: **http://localhost:5173** (или 5174, если 5173 занят).

- Логин и пароль — из `.env` (ADMIN_LOGIN, ADMIN_PASSWORD) или выведены при первом `npx prisma db seed`.

---

## Шаг 4. Привязать Telegram

1. В меню слева нажмите **«Настройки»**.
2. Включите галочку **«Включить уведомления в Telegram»**.
3. Нажмите **«Запросить код привязки»**.
4. На странице появится **код** (например, `ABC123`) и **ссылка на бота** (например, @YourBot).
5. Откройте бота в Telegram (по ссылке или найдите по имени).
6. В чате с ботом отправьте команду: **`/start КОД`** (подставьте ваш код, например: `/start ABC123`).
7. Бот ответит: «Аккаунт привязан к FireNotify...».
8. В приложении нажмите **«Сохранить настройки»**.

Готово. Ваш аккаунт привязан к Telegram.

---

## Шаг 5. Проверить уведомления

- Создайте новое пожарное уведомление: **«Уведомления»** → кнопка добавления (или через API).
- Администраторы с включённым Telegram и привязанным аккаунтом получат сообщение в бота.

---

## Если что-то не работает

| Проблема | Что проверить |
|----------|----------------|
| В логах нет «Bot started» | В `backend/.env` есть строка `TELEGRAM_BOT_TOKEN=токен`, без пробела перед `=`. Перезапустили backend после изменения .env. |
| Бот не отвечает на /start КОД | Код вводите без кавычек, одним сообщением: `/start ABC123`. Код действует до следующего запроса кода. |
| Уведомления не приходят в Telegram | В Настройках включена галочка «Включить уведомления в Telegram», аккаунт привязан (бот ответил «Аккаунт привязан»), вы под админом (admin). |
